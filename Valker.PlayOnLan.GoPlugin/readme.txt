Описание модели.

Центральным объектом является "движок" - объект, реализующий интерфейс IEngine.
В его обязанности входит представление необходимых данных для view model.

Единственным средством воздействия является "ход" - move.
Ход должен быть сделан в соответствии с правилами, за этим следит проверяльщик правил
В результате хода изменяется наблюдаемое состояние игры: ячейки, счёт, текущий игрок
