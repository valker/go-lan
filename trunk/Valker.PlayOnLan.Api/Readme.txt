Valker.PlayOnLan.Api

This project describe API between other components of the system

Необходимо разработать список сообщений, который должен поддерживаться сервером.

Сообщения от клиента
1. Получить список поддерживаемых игр ()
2. Подать заявку на создание игры (имя, тип игры)
3. Получить список заявок
4. Снять заявку (имя, тип игры)
5. Принять чужую заявку (...)

Сообщения от сервера
1. Список поддерживаемых игр (тип игры, ...)
2. Подтверждение поданной заявки (имя, тип игры, статус заявки)
3. Список поданных заявок (заявка, ...)
4. Подтверждение снятия заявки (имя, тип игры, статус заявки)
5. Подтверждение принятия заявки (имя1, имя2, тип игры, статус заявки)
6. Оповещение о начале игры (имя1, имя2, тип игры, ???)
